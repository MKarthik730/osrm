How LAN mode works

1. Run FastAPI on your laptop:
uvicorn main:socket_app --host 0.0.0.0 --port 8000

2. Find your laptop's WiFi IP:
ip addr / ipconfig

3. Enter that IP in Network Mode → LAN

4. Both phones connect to same server over WiFi

5. No internet tunnel needed ✓
